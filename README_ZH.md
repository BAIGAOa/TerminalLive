## 这是什么？

**Terminal Life** 是一款基于终端的人生模拟游戏，使用 [Ink](https://github.com/vadimdemedes/ink) 和 React 构建，采用依赖注入架构，提供丰富的键盘驱动交互界面。

## 环境要求

- **Node.js** >= 18
- 支持 Unicode 和 256 色的终端

## 安装

```bash
npm install -g terminal-live
```
## 使用
```bash
terminal-live
```
