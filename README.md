## What is this?

**Terminal Life** is a terminal-based life simulator built with [Ink](https://github.com/vadimdemedes/ink) and React. It features a dependency injection architecture and rich keyboard-driven UI.

## Requirements

- **Node.js** >= 18
- A terminal with Unicode and 256-color support

## Installation

```bash
npm install -g terminal-live
```

## Usage
```bash
terminal-live
```
